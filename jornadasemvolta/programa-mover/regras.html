Faça parte do MOVER 🚀

A Jornada Sem Volta oferece para você o Programa Mover, de acordo com os seguintes requisitos, que podem ser atualizados sem aviso prévio. Você pode conferir a versão mais recente, a qualquer momento, basta atualizar esta página.

Leia com atenção os requisitos abaixo para participar do Programa Mover.

Regras

    chevron_right

  • A participação é voluntária e não possui nenhum custo ao participante;

    • chevron_right

  • O Link de indicação é personalizado, possibilitando o rastreio de quem o utilizou;

    • chevron_right

  • O participante poderá divulgar o seu link na internet e em perfil de redes sociais podendo, mas não se limitando, a Facebook, Instagram, Youtube, Twitter, LinkedIn, blogs, sites pessoais, etc.;

    • chevron_right

  • Para se inscrever no Programa Mover, basta clicar no botão "QUERO PARTICIPAR" que você será redirecionado para o formulário de inscrição;

    • chevron_right

  • O formulário de inscrição é bem simples, com pouquíssimas informações, tendo como objetivo principal, somente para controle interno;

Como funciona?

    chevron_right

  • Para cada acesso que possuir pelo seu link de indicação, contabilizará 1 ponto;

    • chevron_right

  • Acumulando 10 pontos, contabilizará 1 ficha para o sorteio do prêmio;

    • chevron_right

  • Para cada compra de ingresso do envento com o seu link de indicação, contabilizará 1 ficha para o sorteio do prêmio;

    • chevron_right

  • Quanto maior for a quantidade de ficha que possuir, consecutivamente maior é a probabilidade de chance de vecê ganhar o serteio dos premios;

  • Ou seja:

    • chevron_right

  • 1 acesso = 1 ponto;

    • chevron_right

  • 10 pontos = 1 ficha;

    • chevron_right

  • 1 compra com seu link = 1 ficha;

Premiação

    chevron_right

  • Uma linda caneca personalizada do evento (exclusivo);

    • chevron_right

  • Uma linda almofada personalizada do evento (exclusivo);
